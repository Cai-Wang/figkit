# 期刊图件规范调研汇总

## Nature 系列

| 项目 | 规范 |
|------|------|
| 单栏宽 | 89 mm |
| 双栏宽 | 183 mm |
| DPI | 彩色/灰度 300，纯线图也可 300 |
| 字体 | Helvetica、Arial（sans-serif） |
| 字号 | 正文 5-7 pt，面板标签 8 pt |
| 面板标签 | 粗体小写 (a, b, c) |
| 格式 | AI, EPS, PDF, SVG |
| 色彩模式 | RGB |
| 避讳 | 红-绿组合 |

## Science 系列

| 项目 | 规范 |
|------|------|
| 单栏宽 | 8.5 cm |
| 双栏宽 | 17.5 cm |
| DPI | 彩色 150-300，线图 300 |
| 字体 | Times New Roman |
| 字号 | 6-8 pt |
| 面板标签 | 大写字母 (A, B, C) |
| 格式 | PDF, EPS 首选 |

## Elsevier 系列（Lithos / Chemical Geology 等）

| 项目 | 规范 |
|------|------|
| 单栏宽 | 90 mm |
| 双栏宽 | 190 mm |
| DPI | 半色调 300，组合图 500，纯线图 1000 |
| 字体 | 不限，推荐 TNR 或 Arial |
| 字号 | 正文 7 pt，上下标 >= 6 pt |
| 格式 | PDF, EPS, TIFF |
| 色彩模式 | RGB |

## Lyell Collection（地质学会出版）

| 项目 | 规范 |
|------|------|
| 字号 | 正文 9 pt（最大 12 pt），上下标 >= 6 pt |
| 格式 | 矢量格式优先 |

## 共性地化图惯例

| 图型 | 惯例 |
|------|------|
| REE | 纯线图（无标记）或极小圆点，对数 Y，y=1 灰色虚线 |
| 蛛网图 | 纯线图无标记，29 元素错位标签 |
| TAS | 填充圆点（45-60），加黑色细边 |
| 判别图 | 填充圆点加细边，便于黑白打印 |
| 三元图 | 点大小约为二元图的 1/3-1/2 |
| 调色板 | 低饱和度学术色，避红绿 |
