# A4Grid Cell 居中计算公式

## 目标

在 A4 纸（210×297mm）上放置 `rows×cols` 个等大 cell，垂直居中。

## 公式

```
cell_h = 目标高度（mm）
总高   = rows × cell_h + (rows-1) × hspace
top    = (297 - 总高) / 2
bottom = 297 - 总高 - top
```

## 常见配比

### 1×2（一对 REE+Spider，75mm 宽）

| 目标 cell_h | top/bottom | 说明 |
|-------------|-----------|------|
| 85mm | 106/106 | 近方图（上回误用的尺寸） |
| 50mm | 123/124 | 宽高比 1.5:1，适合蜘蛛网+REE |
| 42mm | 128/128 | 狭长图，适合 8pt 字号 |

### 2×2（两对 REE+Spider，hspace=10）

| 目标 cell_h | 总高 | top/bottom | 说明 |
|-------------|------|-----------|------|
| 85mm | 180mm | 58/59 | 近方图 |
| 50mm | 110mm | 93/94 | 宽高比 1.5:1 |
| 42mm | 94mm | 101/102 | 狭长 |

### Cell 宽度

```
cell_w = (210 - left - right - (cols-1) × wspace) / cols
```

标准边距 left=25, right=25：

| cols | wspace | cell_w |
|------|--------|--------|
| 2 | 10mm | 75mm |
| 2 | 5mm | 77.5mm |
| 2 | 12mm | 74mm |

## 注意

- 裸图预览不需要这套公式——裸图用 `figsize=(8, 5)` 等方便观看的比例
- 这套公式只用于输出到 A4 纸的拼版
